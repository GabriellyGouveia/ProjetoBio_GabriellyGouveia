# 📌 Projeto Bio - Portfólio Pessoal  

## 📖 Sobre o Projeto  

Este projeto consiste no desenvolvimento de um site responsivo do tipo Mini Biografia (Bio), elaborado como atividade da disciplina de Programação Web e Mobile do curso de Ciência da Computação (UNICAP).  

O objetivo principal é organizar e apresentar, de forma estruturada, informações relacionadas à trajetória acadêmica, habilidades e projetos desenvolvidos ao longo da graduação, utilizando tecnologias fundamentais do desenvolvimento web.  

A aplicação foi construída com HTML, CSS e JavaScript, mantendo a separação entre estrutura, estilização e comportamento, conforme proposto na especificação do projeto.  

---

## 🧩 Estrutura do Projeto  

O projeto segue a seguinte organização de arquivos:  

```
/projeto-bio/
│
├── index.html
├── sobre-mim.html
├── habilidades.html
├── projetos.html
├── contato.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── img/
```

---

## ⚙️ Tecnologias Utilizadas  

- HTML  
- CSS  
- JavaScript  

---

## 📱 Funcionalidades  

- Estrutura semântica utilizando `header`, `section` e `footer`  
- Layout responsivo com uso de Flexbox e Media Queries  
- Estilização personalizada com efeitos de hover e transições  
- Menu de navegação responsivo (menu hamburguer)  
- Carrossel de projetos com navegação por botões  
- Exibição automática do ano atual no rodapé com JavaScript  

---

## 📄 Seções do Site  

- **Início:** apresentação inicial  
- **Sobre Mim:** biografia, curso e interesses  
- **Habilidades:** competências técnicas e comportamentais  
- **Projetos:** descrição de projetos com links  
- **Contato:** formulário para envio de mensagens  
- **Footer:** redes sociais e informações adicionais  

---

## 🎯 Considerações  

O desenvolvimento deste projeto permitiu a aplicação prática de conceitos fundamentais de desenvolvimento web, como organização de arquivos, responsividade e interatividade com JavaScript.  

Além disso, o site funciona como um registro da evolução acadêmica e prática ao longo do curso.
